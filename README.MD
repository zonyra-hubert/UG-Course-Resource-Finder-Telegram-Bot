Project DescriptionThe UG Course & Resource Finder Bot is an open-source Telegram bot designed to streamline access to critical academic information for students at the University of Ghana (UG). Built using Python, Django, and the pyTelegramBotAPI library, the bot provides a quick, user-friendly interface to query course details and retrieve associated study materials directly via Telegram.Key FeaturesDatabase Integration (Django): Stores structured course data (codes, titles, credits, lecturer emails) in a robust, scalable Django PostgreSQL database.Course Lookup: Processes user commands (e.g., /find DCIT203) to instantly return detailed course information.Direct Resource Delivery: Uses Telegram's send_document functionality to provide direct links to syllabi, reading lists, or study notes for specific courses.Efficient and Accessible: Offers a low-bandwidth, mobile-friendly solution for academic support, bypassing traditional web portal complexity.Technical StackComponentTechnologyPurposeBackend FrameworkDjango / PythonProvides the data model, database management, and administrative interface.Bot InterfacepyTelegramBotAPI (telebot)Handles message polling, routing, and communication with the Telegram API.Data StoragePostgreSQL / SQLiteStores all course metadata and resource URLs.Environmentpython-dotenvSecurely manages sensitive credentials (BOT_TOKEN) locally.How to Run LocallyClone the Repository: git clone [Your Repository URL]Install Dependencies: pip install -r requirements.txtSetup Environment: Create a .env file and add your BOT_TOKEN.Database Migration: python manage.py makemigrations && python manage.py migrateRun the Bot: python bot.py (Ensure you have populated the database via the Django Admin first!
